# 06_ERP_Manager (ERP 供应链后台 - 开发上下文)
---
### 🎯 模块定位 (Module Scope)
本目录是“数据供应链”核心引擎，负责从外部获取资源并将其标准化。联动 `@02_Global_Business_Flow.md` 中的【采集系统】、【聚合系统】与【报价系统】。
---
### 📂 子菜单核心索引 (Local Routing)
- `ERP_Info_*` (信息中心): 维护平台档案、城市与全局加价公式。
- `ERP_Raw_*` (平台资源): 存储从各大 OTA/集团采集回的异构（Raw）数据。
- `ERP_Map_*` (匹配管理): 核心业务逻辑！处理异构数据向标准数据的映射清洗。
- `ERP_Master_*` (畅行资源): 生成标准可售卖的主数据库（畅行酒店/房型/价态）。
- `ERP_Dev_*` (开发管理): 管控 API 密钥、白名单及底层交易模型。
---
### 🛑 核心开发约束 (Cursor Rules)
- **数据流向闭环**: 严禁直接使用 `Raw` 数据进行售卖，所有业务操作必须基于清洗后的 `Master` 畅行数据。
- **公式剥离**: 在编写报价计算逻辑时，加价公式必须从 `ERP_Info_Price_Rule` 中动态读取，严禁将加价逻辑硬编码（Hardcode）在代码中。
