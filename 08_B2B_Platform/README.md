# 08_B2B_Platform (B2B 分销买家平台 - 开发上下文)
---
### 🎯 模块定位 (Module Scope)
本目录面向下游 C 端/小 B 端用户，是资源分销的前台展示与采购门户。属于 `@01_Glossary_&_Roles.md` 中定义的【交易系统】入口端。
---
### 📂 子菜单核心索引 (Local Routing)
- `B2B_Shop_Hotel` (酒店选购): 前台资源搜索与报价展示。
- `B2B_Order_Center` (订单中心): 买家视角的交易链路流转。
- `B2B_Asset_Center` (账户中心): 预付款余额充值与流水。
- `B2B_Fin_Bill` (财务对账): 买家视角的月结/周结账单。
- `B2B_Dev_Portal` (开发者管理): 对接 API 文档及凭证获取。
---
### 🛑 核心开发约束 (Cursor Rules)
- **只读展示原则**: 前台展现的酒店资源数据绝大多数为只读状态，其数据必须来源于 `06` 模块生成的“畅行标准数据”。
- **预付扣减校验**: 买家产生交易时，必须优先使用事务（Transaction）锁扣减 `Asset_Center` 的预付款余额，防止超卖或资损。
