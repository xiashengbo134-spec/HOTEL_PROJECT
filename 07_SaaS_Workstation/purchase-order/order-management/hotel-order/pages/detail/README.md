# 酒店订单详情页原型

## 页面定位

展示单笔酒店订单的总览、采购过程、金额状态与处理动作。

## 当前页面文件

- `index.html`：主页面版详情页，偏总览与作业
- `side-drawer.html`：抽屉版详情页，偏列表内快速查看

## 当前主页面版特征

- 首屏订单总览卡片
- SLA 剩余时间与风险提示
- 订单与入住信息
- 按日期聚合的采购过程
- 订单轨迹时间线
- 金额与结算摘要
- 状态摘要
- 底部固定操作栏

## 当前抽屉版特征

- 订单主信息摘要
- 多张采购单详情卡片
- 失败原因弹层
- 价格明细弹层
- 钟点房字段展示

## 当前相关操作

主页面版中已出现：

- 返回列表
- 查看日志
- 接单
- 拒单
- 人工补采
- 发起售后
- 确认采购结果
