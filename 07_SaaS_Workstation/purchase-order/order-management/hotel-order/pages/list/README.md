# 酒店订单列表页原型

## 页面定位

酒店订单统一查询、任务分流与快速处理入口页面，是当前模块主工作台。

## 当前主文件

- `index.html`

## 当前页面特征

- 卡片式订单列表，不是传统表格
- 顶部多维筛选区
- Tab 任务分流
- 待处理 SLA 倒计时
- 待处理支持按剩余处理时间排序
- 订单详情抽屉
- 日志抽屉
- 导出 / 导出记录入口

## 当前已展示的核心信息

- 系统订单号 / 外部订单号
- 渠道与订单来源
- 酒店名称、地址、房型、早餐
- 入住人、联系人、入离时间
- 支付金额、结算金额、售卖模式
- 系统状态、平台状态、采购状态、售后状态
- 预订时间、支付时间、取消时间

## 当前操作

- 接单
- 拒单
- 订单详情
- 日志
